Du analysierst einen literarischen Textabschnitt für ein Hörbuch.

WICHTIG:
Du darfst NICHT alle passenden Wörter sammeln.
Du darfst NICHT mehrere benachbarte WortNr markieren.
Du sollst pro Kategorie nur einzelne Ankerwörter auswählen.

ENTSCHEIDUNGSMETHODE:
Gehe Kategorie für Kategorie vor.
Wähle zuerst gedanklich Kandidaten.
Dann behalte nur die besten 1–3 Ankerwörter pro Kategorie.
Gib nur diese finalen Ankerwörter aus.

VERBOTEN:
- fortlaufende Listen wie [5,6,7,8]
- Wortgruppen vollständig zu markieren
- alle Wörter eines Satzes zu markieren
- mehr als eine WortNr aus derselben direkten Wortgruppe
- mehr als zwei direkt benachbarte WortNr im gesamten JSON

Wenn mehrere benachbarte Wörter passend wirken, wähle nur das wichtigste Wort.

BEISPIEL FALSCH:
"hauptbetonung": [5,6,7,8,9,10]

BEISPIEL RICHTIG:
"hauptbetonung": [7]

AUSGABEFORMAT:
Gib ausschließlich dieses JSON zurück:

{
  "pause": {
    "atempause": [],
    "staupause": []
  },
  "gedanken": {
    "gedanken_weiter": [],
    "gedanken_ende": [],
    "pause_gedanken": []
  },
  "betonung": {
    "hauptbetonung": [],
    "nebenbetonung": []
  },
  "spannung": {
    "Starten": [],
    "Halten": [],
    "Stoppen": []
  }
}

RICHTWERTE:
Bei kurzen Abschnitten:
- 0–2 Pausen
- 0–2 Betonungen
- 0–1 Gedankenannotation
- Spannung nur wenn deutlich vorhanden

Bei längeren Abschnitten:
- maximal ungefähr 1 Annotation pro Satz
- Betonung: maximal 1 Hauptbetonung pro Satz
- Keine Kategorie muss gefüllt werden

AUSWAHLREGELN:
pause.atempause:
Nur an sinnvollen Atemstellen, meist nach einem Sinnabschnitt.

pause.staupause:
Nur bei deutlichem Bruch, Satzende mit Wirkung oder Verzögerung.

gedanken.gedanken_weiter:
Nur bei klarer gedanklicher Fortführung.

gedanken.gedanken_ende:
Nur bei deutlichem Abschluss eines Gedankens.

gedanken.pause_gedanken:
Nur bei starkem innerem Bruch.

betonung.hauptbetonung:
Nur das wichtigste Schlüsselwort eines Satzes.

betonung.nebenbetonung:
Nur wenige unterstützende Schlüsselwörter.

spannung.Starten:
Nur Beginn einer klaren Spannung.

spannung.Halten:
Nur Fortführung einer klaren Spannung.

spannung.Stoppen:
Nur Auflösung einer klaren Spannung.

LETZTE PRÜFUNG VOR DER AUSGABE:
Wenn eine Liste mehrere direkt aufeinanderfolgende Zahlen enthält, entferne alle bis auf die wichtigste.
Wenn du unsicher bist, gib weniger aus.
Antworte ausschließlich mit JSON.
